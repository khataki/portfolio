CodingTimer

Описание проекта: CodingTimer — это удобный инструмент для разработчиков, созданный с целью отслеживания времени, затраченного на изучение различных инструментов и языков фронтенд-разработки. Этот проект идеально подходит для тех, кто хочет вести учет своих учебных часов и анализировать, как они распределяются между разными технологиями.

Функциональность:

  • Запоминание данных: система сохраняет введенные данные, что позволяет пользователям продолжать отслеживать свои результаты даже после закрытия приложения.
  • Поддержка популярных языков: возможность выбора из предустановленных языков программирования или добавление собственного.
  • Аналитика времени: приложение ведет учет количества часов, потраченных на изучение каждого языка программирования ежедневно и в целом.
  • Визуализация данных: приложение использует Charts.js для создания наглядных графиков.

Планы на будущее: Добавление многоязычной поддержки и встроенного таймера для отслеживания времени в реальном времени.

Технологии: HTML, CSS, JavaScript, Charts.js

Калькулятор фурнитуры для стеклянных козырьков

Описание проекта: Калькулятор фурнитуры для стеклянных козырьков — это специализированный инструмент, предназначенный для расчета необходимого количества и типа фурнитуры при установке стеклянных козырьков.

Функциональность:

  • Ввод данных: пользователи могут вводить размеры козырька, вес стекла и другие параметры, необходимые для расчета.
  • Расчет фурнитуры: калькулятор автоматически рассчитывает количество необходимой фурнитуры на основе введенных данных.
  • Подбор фурнитуры: программа предоставляет рекомендации по выбору конкретных типов фурнитуры, учитывая характеристики стекла и архитектурные особенности.
  • Интерактивные подсказки: встроенные подсказки помогают пользователям вводить корректные данные и понимать, какие параметры влияют на конечный расчет.

Планы на будущее: Поддержка различных типов козырьков, интеграция с базой данных поставщиков и экспорт результатов расчета.

Технологии: HTML, CSS, JavaScript, Node.js, Express.js, jQuery

MemoryDesk

Описание проекта: MemoryDesk — это интерактивное веб-приложение, созданное в память о моем прадеде, ветеране Второй мировой войны. Проект использует технологию Mapbox для визуализации ключевых мест, связанных с его жизнью и военными подвигами.

Функциональность:

  • Интерактивная карта: приложение отображает карту с использованием Mapbox, где отмечены значимые локации, связанные с жизнью моего прадеда.
  • Информация о локациях: при клике на метки на карте пользователи могут увидеть подробную информацию о каждом месте, включая исторические факты и фотографии.
  • Анимированная маршрутизация: визуализация маршрутов, которые проделал мой прадед во время своей службы, с использованием анимаций.

Планы на будущее: Добавление новых локаций и мультимедийных материалов, а также возможность пользователям добавлять свои семейные истории.

Технологии: HTML, CSS, JavaScript, Mapbox, Leaflet.js

Элария: Войны фракций

Описание проекта: Элария: Войны фракций — это стратегическая игра в жанре RTS, где игроки оказываются в мире на грани разрушения, выбирая свою фракцию и участвуя в эпических битвах за власть и ресурсы.

Основные особенности версии 1.0:

  • Октагональные клетки: уникальная структура карты для стратегического планирования.
  • Режим RTS: принятие решений в реальном времени.
  • Рандомная генерация карты: каждое прохождение предлагает уникальную карту.
  • Разные фракции: Люди, Эльфы, Гномы, каждая с уникальными лидерами и способностями.
  • Прокачка юнитов: развитие юнитов для адаптации к изменяющимся условиям боя.
  • Постройка сооружений: укрепление позиций и обеспечение ресурсами.
  • Битвы между юнитами: тактические сражения, определяющие исход игры.
  • Население и расы: рост населения зависит от количества пищи.
  • Торговля: возможность обмена ресурсами между фракциями.

Планы на будущее: Добавление новых фракций, расширение мира и введение глобальных событий.

Технологии: Игровой движок, графика и анимация, AI для управления враждебными фракциями.
